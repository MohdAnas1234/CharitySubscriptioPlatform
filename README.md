# Golf Charity Subscription Platform

Full MERN stack project scaffold for a premium golf charity subscription platform.

- React + Vite frontend
- Tailwind CSS
- React Router DOM
- Axios
- Framer Motion
- Node.js + Express backend
- MongoDB + Mongoose
- JWT auth
- bcryptjs
- Multer
- Cloudinary optional support
- Vercel frontend deployment
- Render backend deployment

## Structure

- `client/` — frontend app
- `server/` — backend API
